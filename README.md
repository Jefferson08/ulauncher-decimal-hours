# Decimal Hours Converter (Ulauncher)

Converte `minutos/HH:MM` para horas decimais e horas decimais para `HhMm`.

## Keywords padrão

- `hd` `minutos/HH:MM` -> decimal
- `hm` decimal -> `HhMm`

## Exemplos

- `hd 46m`
- `hd 1:30`
- `hd 1h20m`
- `hd 90`
- `hm 1.56`
- `hm 1.5+0.25`
