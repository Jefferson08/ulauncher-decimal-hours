# Decimal Hours Converter (Ulauncher)

Converte horas decimais para `HH:MM` e `HH:MM/minutos` para decimal.

## Keywords padrão

- `hd` decimal -> `HH:MM`
- `hm` `HH:MM/minutos` -> decimal

## Exemplos

- `hd 1.75`
- `hd 1.5+2.25+0.75`
- `hd (1.5*2)+0.25`
- `hm 1:30`
- `hm 1h20m`
- `hm 2h`
- `hm 90m`
- `hm 90`
